Organizing your objects

 

NotreDAM - Open Source Digital Asset Management - developed by CRS4 and Sardegna Ricerche (DistrICT Lab)

One of the most powerful tools for organizing objects is the Catalogue.

The catalogue includes several tools, such as the Keywords, the Inbox, the Collections, etc..

The Keywords provides a taxonomy view of classification tags that can be easily mapped on XMP Metadata.

The following figure shows an example of Keywords:




The user can add new keywords and re-organize their hierarchical structure. Each workspace has its own catalogue, so the taxonomy of keywords can be customized for each workspace. In order to add a new keyword,  you have to right click on the parent keyword and select Add keyword. The following example shows how to add the keyword Barcelona and map it to the dc:subject property:




By clicking Save, the keyword Barcelona will be added to the taxonomy. The “associate ancestors” option shall be set to automatically associate to the objects also the ancestors of keyword Barcelona (in this case, Spain and Europe).


In the following example, we want to associate the keyword Seville to a set of pictures.




We can perform this operation by selecting the pictures and then clicking the checkbox on the left of the keyword:




Note that we associated the keyword Seville only. The other two keywords have been automatically associated because we chose the “associate ancestors” option at keyword creation.


In a similar way, you can add objects to Collections. Collections are containers of objects and, unlike keywords, do not involve any association of metadata.
