NotreDAM

The open-source Digital Asset Management platform

 
 

Installation by packages

 

NotreDAM - Open Source Digital Asset Management - developed by CRS4 and Sardegna Ricerche (DistrICT Lab)

System requirements


To install NotreDAM server you need at least one computer running Ubuntu 9.10. On this computer, you have to install NotreDAM packages (as described below).

You also need at least one MediaDART node. Although MediaDART nodes are expected to be installed on different computers, a minimal installation (for evaluation purpose) can be done installing the MediaDART node on the same computer running the NotreDAM server.

To install a MediaDART node please refer to the MediaDART website.


A NotreDAM deployment should be done with several MediaDART nodes in order to provide redundant storage and parallel processing.


Please note that you can use a local MediaDART node (i.e. on the same computer running the NotreDAM server) or remote nodes (i.e. on pcs different from the one running the NotreDAM server). By default, NotreDAM is configured to work with a local node. To change this setting, please refer to the advanced configuration page.



Step 1: Install MediaDART nodes


To install a MediaDART node please refer to the MediaDART website.



Step 2: Install the NotreDAM server


To install the NotreDAM server on your computer just add the following lines to your /etc/apt/sources.list file:


deb http://www.mediadart.org/stage-repository/karmic binary-i386/

deb http://www.mediadart.org/stage-repository/karmic binary-amd64/


Then, open a terminal window and enter the following commands:


sudo apt-get update

sudo apt-get install notredam


This commands will install the application under /opt/notredam.



Step 3: Start MediaDART nodes


To start the MediaDART node(s) please refer to the MediaDART website.



Step 4: Start the NotreDAM server


Open a terminal window and enter the following commands:


cd /opt/notredam

python notredam.py -- runserver run



At this point you can start using NotreDAM from a browser, as described in the Quick Guide page.
