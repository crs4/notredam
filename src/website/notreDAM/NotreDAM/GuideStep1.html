After the installation, you can access NotreDAM with your default browser at the following url :


http://server_ip_address:8000/


where server_ip_address is the IP address of the NotreDAM server (if you are running the server on your local machine, just use the url http://127.0.0.1:8000/) 


A login screen will appear.




By default the user login is:

User: demo

Password: demo


You can also register a new user by clicking “New user? register now!” link on the login screen and enter the required information.


After a successfull login, you will enter the Workspace View, explained in the next step.
